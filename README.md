# Activity 3

This project is a simple **ASP.NET MVC front-end application** created for **Activity 3**.  
It is a **fan-made website inspired by the game _Love and Deepspace_**, focused on practicing UI design, forms, and layout using C#.

---

## 🌟 Features
- Home page with fan content  
- Fan interaction form (name, character, message, rating)  
- Display of submitted entries in a table  
- Basic styling using CSS and Bootstrap  

---

## 🛠️ Tech Stack
- C#  
- ASP.NET MVC  
- Razor Views  
- Bootstrap
