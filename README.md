# Activity 3 & 4

This repository contains two activities combined in one project for learning and practicing ASP.NET MVC development.

It is a **fan-made website inspired by the game _Love and Deepspace_**.

- **Activity 3** -> main branch
- **Activity 4** -> Activity-4 branch

---

## 🌟 Project Overview
### Activity 3 (Main Branch)
A simple ASP.NET MVC front-end application focused on:

- UI design
- Forms
- Page layout
- Basic data handling using an in-memory database

The project is a fan-made website inspired by the game Love and Deepspace for academic purposes.

### Activity 4 (Branch: Activity-4)
An extended version of the project where the application is connected to a real database using SQL Server (SSMS) for persistent data storage and database operations.

---

## 🌟 Features
- Home page with fan content 
- Fan interaction form (name, character, message, rating)
- Display of submitted entries in a table
- In-memory database (Activity 3)
- SQL Server database integration (Activity 4)
- Basic styling using CSS and Bootstrap

---

## 🛠️ Tech Stack
- C#  
- ASP.NET MVC  
- Razor Views  
- Bootstrap  
- SQL Server Management Studio 2022
